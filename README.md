# NodeJS

Installation :

sudo npm install -g


Commandes :


Voir toutes les caracteristiques d'un champion :

node app.js
Puis entrez le nom d'un champion (ex : 1001)

Afficher toutes les versions du jeu :

node app.js -ver 



Voir toutes les caracteristiques d'un item :
node app.js -i

Puis entrez l'ID de l'item (ex : 1001)
